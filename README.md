# android2020
android project repository

FoodFinder app basic Android demo application in which RoomDatabase, Glide Api and RecycleViewer was used.

implements functions:
		- add Restaurant
		- update Restaurant
		- delete Restaurant
		- list all Restaurant
		- update Profile
		
Project structure: 
	-Activity : 
      - MainActivity
	-Fragments:
      -AddRestaurant
      -EditProfile
      -RestaurantList
      -RestaurantDetail
 -Database Classes:
    -FoodDatabase
    -RetaurantDAO
    -ProfileDAO
